# globo
Take me somewhere I can dream


## The project:
Our goal was to make an flight searching tool with a globe User Interface. The particularity of our project is you search entirely based on price, and not at all on destination. At Globo, our motto is "Take me somewhere I can dream", you can write a description of wherever you want to go with a price maximum and Globo will find the flights and render them on the globe UI.

### Description:
Globo provides a minimal search interface allowing you to search spontaneously, limited only by a budget and a dream. Using our AI search bar, your search can be as pragmatic as "a beachside getaway" or as abstract as "somewhere I can bask in warmth"; with Globo, the globe is your oyster.

### Challenges:
The main challenge was the complete absence of any free API to track flight data. We had to bypass this with unconventional methods which we'll explain orally. Furthurmore, the globe interface was very hard to control with React, making any change to the HTML page a true challenge.

### Reflections:
We are very happy of what we could accomplish in 48 hours, we learnt incredibly more than in any computer science course and we are looking forward to use these newly acquired skills on future projects.
