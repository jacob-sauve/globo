# globo
Take me somewhere I can dream



## The project:
Our goal was to make an flight searching tool with a globe User Interface. The particularity of our project is you search entirely based on price, and not at all on destination. At Blobo, our motto is "Take me somewhere I can dream", you can write a description of wherever you want to go with a price maximum and Blobo will find the flights and render them on the globe UI.

## Challenges:
The main challenge was the complete absence of any free flight data api. We had to bypass this with unconventional methods which we'll explain orally. Furthurmore, the Globe interface was very hard to control with React, making any change to the html page a true challenge.

## Reflections:
We are very happy of what we could accomplish in 48 hours, we learnt incredibly more than in any computer science course and we are looking forward to use these newly acquired skills on future projects.
